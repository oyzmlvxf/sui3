入门

CDN 地址

我们每一个版本都会发布到阿里的CDN服务器上，不仅访问迅速，而且我们会保留每一个版本。你不用担心我们更新之后导致代码出错或者找不到老版本的链接。

<!--下面的CSS放在 HEAD 中-->
<link rel="stylesheet" href="//g.alicdn.com/sui/sui3/0.0.18/css/sui.min.css">
<!--下面的JS放在 body 的最后-->
<script type="text/javascript" src="//g.alicdn.com/sj/lib/jquery/dist/jquery.min.js"></script>
<script type="text/javascript" src="//g.alicdn.com/sui/sui3/0.0.18/js/sui.min.js"></script>

如果你是在开发或者测试阶段，希望能调试源码，可以引用非压缩的代码。

<!--下面的CSS放在 HEAD 中-->
<link rel="stylesheet" href="//g.alicdn.com/sui/sui3/0.0.18/css/sui.css">
<!--下面的JS放在 body 的最后-->
<script type="text/javascript" src="//g.alicdn.com/sj/lib/jquery/dist/jquery.js"></script>
<script type="text/javascript" src="//g.alicdn.com/sui/sui3/0.0.18/js/sui.js"></script>

如果你希望组件都带有 sui- 前缀，可以这样引用：

<!--下面的CSS放在 HEAD 中-->
<link rel="stylesheet" href="//g.alicdn.com/sui/sui3/0.0.18/css/sui-prefixed.min.css">
<!--下面的JS放在 body 的最后-->
<script type="text/javascript" src="//g.alicdn.com/sj/lib/jquery/dist/jquery.min.js"></script>
<script type="text/javascript" src="//g.alicdn.com/sui/sui3/0.0.18/js/sui-prefixed.min.js"></script>

ie8 兼容性

为了使在IE8中响应式栅格可以正常工作，以及可以使用HTML5的标签，你需要额外引入两个文件：

<!--[if lt IE 9]>
<script src="//g.alicdn.com/sui/sui3/0.0.18/js/lib/html5shiv.min.js"></script>
<script src="//g.alicdn.com/sui/sui3/0.0.18/js/lib/respond.min.js"></script>
<![endif]-->

下载

建议直接引用我们的CDN地址，保证稳定快速的访问，而不是把文件下载到本地或者传到你自己的服务器。

SUI 是开源项目，你可以直接从github上克隆我们的源代码并自行编译。地址： SUI3 地址

页面模板

你可以直接使用下面的模板作为你的页面基本结构。

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="utf-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1">
    <title>SUI 模板</title>
    <link rel="stylesheet" href="//g.alicdn.com/sui/sui3/0.0.18/css/sui.min.css">
  </head>
  <body>
    <h1>Hello, world!</h1>
    <p>这里换成你自己的内容</p>
    <script type="text/javascript" src="//g.alicdn.com/sj/lib/jquery/dist/jquery.min.js"></script>
    <script type="text/javascript" src="//g.alicdn.com/sui/sui3/0.0.18/js/sui.min.js"></script>
  </body>
</html>

从旧版本SUI升级

如果你的产品中使用了老版本的SUI，我们强烈建议升级到新版本。因为这次升级并不能完全兼容旧版，所以升级过程中你可能会碰到一些问题。下面给出了我们的升级指南，和常见问题的解决方法。

图标问题

因为SUI3使用了另一套icon font，如果你是从旧版的SUI升级到SUI3，最好把图标替换为新版图标。我们强烈建议不要继续使用旧版的图标，因为以后不会在更新旧版的SUI，但是如果出于某些原因（比如旧的图标在新版中没有，或者旧的代码不想做修改），那么你可以引用下面的一个CSS文件来使用旧版的图标，旧版SUI图标文档

<!--下面的CSS放在 HEAD 中-->
<link rel="stylesheet" href="http://g.alicdn.com/sui/sui3/0.0.2/old/css/icons.min.css">
