可访问性问题

轮播图并不支持可访问（无障碍）标准，如果你需要兼容的话，请考虑其他方式来实现。

IE 8 & 9不支持动态效果

本组件只使用CSS3作为动画实现，但是Internet Explorer 8 & 9 并不支持必须得CSS属性. 因此, 在这类浏览器中是不存在动态效果的.

需要初始化一个激活元素

.active 需要被添加到一个元素当中, 否则, 轮播图将不会显示.

可选字幕

可以轻松的在轮播元素上添加字幕，添加 .carousel-caption 元素，在.item当中. 几乎可以在其中放置任意html内容，它会将内容自动对其与格式化

用法

多轮播

Carousels require the use of an id on the outermost container (the .carousel) for carousel controls to function properly. When adding multiple carousels, or when changing a carousel's id, be sure to update the relevant controls.

使用data属性

使用 data 属性可以很容易的控制轮播（Carousel）的位置。data-slide 接收关键参数 prev 或者 next, 将会根据轮播元素目前的位置改变其位置。 与此同时, 使用 data-slide-to传递轮播图的原始位置到data-slide-to="2",也将会切换轮播图到响应位置.

data-ride="carousel" 属性被用来标记一个轮播图在页面加载后开始动画它不能和显式的js初始化方式一起使用

使用js

手动调用轮播图:

{% highlight js %} $('.carousel').carousel() {% endhighlight %}

参数

可以通过data属性或者js传递参数,使用data属性，只需把参数名添加到data-, 例如data-interval="".

Name type default description
interval number 5000 轮播图切换的时间间隔，如果没有设置轮播图不会自动循环.
pause string "hover" 在鼠标进入进入轮播区域后自动暂停循环，并在鼠标离开后恢复
wrap boolean true 轮播图是否应该一直循环还是可以结束
keyboard boolean true 轮播图是否响应键盘事件

方法

.carousel(options)

初始化轮播为可选的object 并开始循环项目.

{% highlight js %} $('.carousel').carousel({ interval: 2000 }) {% endhighlight %}

.carousel('cycle')

从左到右循环轮播项目.

.carousel('pause')

停止轮播循环项目.

.carousel(number)

循环轮播到某个特定的帧（从 0 开始计数，与数组类似）.

.carousel('prev')

循环轮播到上一个项目.

.carousel('next')

循环轮播到下一个项目.

事件

轮播插件暴漏了两个与轮播功能相关的事件

两个事件都有以下属性

  • direction:轮播内容的方向 (不是 "left" 就是 "right").
  • relatedTarget:正在滑入的当前激活轮播元素.
Event Type Description
slide 该事件会在调用slide实例方法时立即触发.
slid 该事件会在完成幻灯片过滤效果时触发该事件.
