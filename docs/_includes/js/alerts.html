用法

只需增加data-dismiss="alert"到你的关闭按钮，就回自动拥有一个警告关闭功能，关闭警告框会将其从DOM中移除.

{% highlight html %} {% endhighlight %}

想要让你的警告框在关闭时拥有渐隐，需要在元素上增加 .fade和.in样式类

方法

$().alert()

使警告能够监听拥有data-dismiss="alert"属性的后代元素的单击事件. (在使用data-这类自动初始化没有必要使用.)

$().alert('close')

通过将警告移出DOM关闭警告框. 如果 .fade 和 .in 已在节点上应该，那么早移除前，警告框会有渐隐的效果.

事件

警告框暴漏了一些与警告功能相关的事件可供使用.

Event Type Description
close 该事件会在close实例方法调用后执行.
closed 该事件会在警告框被关闭后调用。（会等待CSS动画结束）
