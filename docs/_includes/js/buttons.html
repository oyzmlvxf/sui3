按钮 button.js

该插件允许你加强按钮的功能。你可以控制按钮的状态，也可以为组件创建按钮组，比如工具条。

跨浏览器兼容性

在页面多次加载之间 Firefox 依旧保持表单控制状态(不可用和不可选). 一种解决方式是设置autocomplete="off". 参见 Mozilla bug #654072.

状态

通过添加 data-loading-text="Loading..."可以为按钮设置正在加载的状态.

使用任何你想用的状态

在此次演示中,我们使用了 data-loading-text 和 $().button('loading'), 但是这不仅仅是你可以使用的状态. 参见更多$().button(string) 文档.

单例切换

增加 data-toggle="button"能够激活单个按钮的切换.

预切换按钮需要 .active 和 aria-pressed="true"

对预切换按钮,你必须增加.active样式和aria-pressed="true" 属性到 button 自身.

复选 / 单选

增加 data-toggle="buttons"到 .btn-group包含复选或者单选的输入框容器可以启用各自组件的样式.

预选择一个选项需要增加.active

想要预选一个内容, 你必须自行增加 .active 样式到一个输入框label

视觉上的选中状态仅能通过click事件更新

如果复选框的选中状态没有通过click事件(e.g. via <input type="reset">或者通过设置输入框的checked 属性, 你将需要自行切换.active输入框上的样式label

方法

$().button('toggle')

切换按压状态。赋予按钮被激活的外观。

$().button('reset')

重置按钮状态，文本内容恢复为最初的内容。

$().button(string)

重置按钮状态，并添加新的内容。
